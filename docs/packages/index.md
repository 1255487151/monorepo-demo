# 库文档

## 分组原则

这一组页面只收录当前 monorepo 中对外发布的库，不承接配置教程，也不承接规范约束文档。

## 当前发布库

| 包名                       | 定位           | 主要内容                                              |
| -------------------------- | -------------- | ----------------------------------------------------- |
| `@smallbrother/components` | Vue 3 组件库   | 组件概览、`XlgSelect`、`XlgTable`、样式入口、resolver |
| `@smallbrother/directives` | Vue 3 指令库   | `table-auto-height` 指令与插件安装入口                |
| `@smallbrother/utils`      | 通用工具函数库 | `add`、`isEmpty`、`deepClone`、`debounce`、`throttle` |
