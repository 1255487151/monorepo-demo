# 配置学习

## 定位

这一组页面用于学习当前仓库的工具链配置，不承担规范约束职责，也不承担发布库 API 说明职责。

## 收录范围

- [ESLint](/learn-config/eslint)
- [Git](/learn-config/git)
- [TSConfig](/learn-config/tsconfig)
- [Vite Config](/learn-config/vite-config)

## 阅读建议

1. 先看 `TSConfig` 了解 monorepo 的类型系统边界。
2. 再看 `ESLint` 理解代码风格和静态检查。
3. 再看 `Vite Config` 理解各包如何打包。
4. `Git` 用于补齐日常协作和仓库提交流程知识。
